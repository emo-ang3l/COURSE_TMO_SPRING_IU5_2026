# Отчет по лабораторной работе №6

## Цель работы
Исследовать задачу условного экстремума (вариант 1.1) с помощью метода множителей Лагранжа и условий KKT.

## Краткая теория
Для задач с ограничениями типа равенств/неравенств применяются условия Куна-Таккера. Они включают стационарность Лагранжиана, допустимость, неотрицательность множителей и условие дополняющей нежесткости. Характер стационарной точки уточняется через условия второго порядка.

## Постановка задачи
Выполнить исследование функции варианта 1.1 при заданных ограничениях:
- составить функцию Лагранжа;
- записать условия первого порядка;
- найти стационарные точки;
- проверить допустимость;
- проверить условия второго порядка;
- вычислить значения функции в найденных точках;
- показать результат графически.

## Что сделано
1. В ноутбуке задана целевая функция и ограничения варианта.
2. Построен Лагранжиан и получены уравнения стационарности.
3. Реализован программный поиск точек KKT через перебор активных ограничений.
4. Для каждой найденной точки выполнены проверки:
   - допустимость ограничений,
   - знаки множителей Лагранжа,
   - условие дополняющей нежесткости.
5. Выполнена проверка второго порядка по матрице Гессе Лагранжиана.
6. Построена графическая визуализация допустимой области, линий ограничений и найденных точек.

## Результаты
В ноутбуке представлены найденные стационарные точки, значения целевой функции, результаты проверки условий KKT и графическое подтверждение решения.

## Выводы
Решение получено программно и подтверждено аналитическими условиями и графикой. Проведенная проверка позволяет обосновать найденные точки условного экстремума.
