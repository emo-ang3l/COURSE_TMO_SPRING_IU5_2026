# Отчет по лабораторной работе №5

## Цель работы
Изучить и сравнить качество ансамблевых моделей для задачи классификации.

## Краткая теория
Ансамблевые методы объединяют несколько базовых моделей для повышения качества и устойчивости предсказаний. Бэггинг уменьшает дисперсию моделей (например, Random Forest, Extra Trees), а бустинг последовательно исправляет ошибки предыдущих моделей (AdaBoost, Gradient Boosting).

## Постановка задачи
Подготовить данные, выполнить разделение на train/test и обучить четыре ансамблевые модели: две из группы бэггинга, AdaBoost и GradientBoosting. Сравнить модели по одной метрике качества.

## Что сделано
1. Использован датасет Breast Cancer Wisconsin.
2. Выполнена базовая предобработка признаков.
3. Данные разделены методом `train_test_split`.
4. Обучены модели: `RandomForest`, `ExtraTrees`, `AdaBoost`, `GradientBoosting`.
5. Рассчитана метрика `f1` на тестовой выборке.
6. Выполнено сравнение качества всех моделей в единой таблице.

## Результаты
В ноутбуке приведена сводная таблица значений `f1` для всех ансамблевых моделей и выделена лучшая модель.

## Выводы
Ансамблевые модели обеспечили высокое качество классификации. Сравнение показало, какая архитектура ансамбля (бэггинг или бустинг) более эффективна на выбранных данных.
